## Event managing app
This is a simple console app for managing events. It shows a menu where user can interact with the app.

## How to use app
To use this app first clone it:
``` javascript
git clone git@github.com:mshirr3/EventTrackingApp.git
```
then download npm:
``` javascript
npm i
```
and start the app:
``` javascript
npm start
```
